# CS260_huntDuck_demo
This is the demo of the study of web design

what I do

# 1. 创建项目 (选择 React + JavaScript)
npm create vite@latest duck-hunt -- --template react

# 2. 进入目录
cd duck-hunt

# 3. 安装 React Router 和 Bootstrap (Bootstrap是该课程常用的样式库)
npm install react-router-dom bootstrap react-bootstrap

# 4. 启动项目
npm run dev
